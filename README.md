# Vido 🎥

Vido is a modern video-sharing platform inspired by YouTube, where users can upload, watch, and explore videos seamlessly. Built with a focus on performance, user experience, and scalability.

---

## 🚀 Features

- 🔐 User Authentication
- 📤 Upload Videos
- 🎬 Watch Videos
- 📱 Responsive Design
- 👤 User Profiles
- ⚡ Fast & Modern UI

---

## 🛠️ Tech Stack

### Video Handling
- Imagekit

### Frontend
- React.js / Next.js
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- NextAuth / JWT
